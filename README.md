# 项目简介 python720

本项目是一款基于Android平台的Python心理健康APP，致力于为大学生提供心理健康服务。以下是项目的相关介绍：

## 技术栈

- **前端：** 安卓原生开发，兼容小程序
- **后端：** 使用Python进行开发
- **数据库：** MySQL

## 功能模块

- 心理健康测试
- 心理咨询预约
- 情绪日记记录
- 心理课程学习
- 成长计划制定

## 系统角色

- 学生用户
- 心理咨询师
- 管理员

## 运行环境

- **前端：** 安卓设备，兼容小程序
- **后端：** 推荐运行在Linux服务器上，Python版本需大于3.6

## 部署步骤

1. 配置Python环境
2. 安装所需依赖
3. 配置数据库
4. 部署后端服务
5. 部署前端应用

## 目录结构

```
项目根目录/
├── backend               # 后端代码目录
│   ├── app               # 应用代码目录
│   ├── config            # 配置文件目录
│   ├── db                # 数据库相关目录
│   └── ...
├── frontend              # 前端代码目录
│   ├── android           # 安卓应用代码
│   └── miniprogram       # 小程序代码
└── ...
```

## 核心亮点

- **跨平台：** 支持Android设备，同时兼容小程序，方便用户使用
- **个性化服务：** 提供多种心理健康相关功能，满足不同用户的需求
- **易于部署：** 提供详细的部署文档，简化部署流程
- **成长计划：** 帮助用户制定个性化的心理健康成长计划，关注用户成长
- **源码+文档：** 提供完整源码和配套文档，便于二次开发和学习交流
